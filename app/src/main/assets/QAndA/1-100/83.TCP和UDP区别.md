|                             TCP                              |                             UDP                              |
| :----------------------------------------------------------: | :----------------------------------------------------------: |

|                          工作效率低                          | 有较好的实时性，效率高，适用于对告诉传输和实时性有较高要求的通信或者广播通信。 |
|                       连接只能是点对点                       |            连接支持一对一、一对多、多对一、多对多            |
