### HashMap 的两种遍历方式

```java
//效率高,以后一定要使用此种方式！只遍历一次,将key及value都放到entry中,效率高
Map map = new HashMap();
Iterator iter = map.entrySet().iterator();
while (iter.hasNext()) {
　　Map.Entry entry = (Map.Entry) iter.next();
　　Object key = entry.getKey();
　　Object val = entry.getValue();
}
```

```java
//效率低,以后尽量少使用！遍历2次,第一次是获取iterator时,第二次是从hashmap中取出key所对应的value.效率低
Map map = new HashMap();
　　Iterator iter = map.keySet().iterator();
　　while (iter.hasNext()) {
　　Object key = iter.next();
　　Object val = map.get(key);
}
```
